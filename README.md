# Voteria
Voteria is a Reddit-inspired community feed built with Go

# Getting Started
You need Golang installed in your system.

Clone the project:
```
git clone github.com/yaghoubi-mn/voteria-backend
```
Change directory:
```
cd voteria-backend
```
Add test data:
```
go run . test-data
```
Run the server:
```
go run .
```
